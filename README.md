# Asharib Khan - 3D Spatial Portfolio

A highly immersive, cinematic 3D spatial portfolio built with Next.js, Framer Motion, and Three.js.

## Tech Stack
- **Framework:** Next.js 15 (App Router)
- **Styling:** Tailwind CSS + Custom Utility Variables
- **3D Rendering:** React Three Fiber (Three.js) & Spline
- **Animations:** Framer Motion (Spatial Scroll-Jacking Layout)
- **Smooth Scrolling:** Lenis

## Features
- **Cinematic Sphere Layout:** A completely custom scroll-jacking engine that mounts every section into the center of the screen, mapping scroll progress to extreme 3D mathematical transformations.
- **WebGL Glass Gallery:** A dynamic 3D background containing perfect glass refractive shapes (TorusKnot, Sphere, Torus) that spin and react to your mouse cursor and scroll position.
- **Ultra-Dark Aesthetic:** Designed with a deeply minimalistic `#050505` background, utilizing glassmorphism and dramatic lighting effects.
- **Liquid Masking:** Optical illusions using radial gradient masks to perfectly blend 3D canvas iframes into the DOM.

## Running Locally

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
